Задание на 2 часа в 17:15 репозиторий закрою:

1. Скачать файлы с репозитория
2. Cоздать проект в VS или в QT
3. Написать комментарии для Doxygen
4. Написать Dockerfile:
	В качестве исходного образа взять 	ubuntu.	
	Разместить информацию об авторе как метку.
	Установить модуль gсс.
	Скопировать файлы из репозитория в контейнер в папку MyProj.
5. Создать докер образ.
6. Cоздать файл readme c последовательностью команд git.
7. Загрузить все в репозиторий в ветку с именем <Фамилия>_<номер группы>.

Результат экзамена:
1. Cозданная ветка, например Budylina_201-345
2. В ветке находятся файлы: 
    Dockerfile, 
    t2.txt, 
    t.txt, 
    main.cpp, 
    DBClass.h, 
    DBClass.cpp 
    файлы проекта *.pro или *.vcxproj, *.sln и *.filters.
    Файл readme (cодержит последовательность всех команд git, которые нужно будет вызвать для работы с репозиторием).
